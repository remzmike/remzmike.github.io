SimpleUI

SimpleUI is an immediate-mode ui library in javascript on the html5 canvas.

I wrote it in c# 10 years ago, ported to lua 5 years ago, and then ported to this javascript.

An 'immediate-mode ui' is designed to integrate with realtime multimedia and games, executing on every frame. That is why I made it. Some people will now recognize it as similar to react or angular because programming with it gives you a hierarchy of functions which are effectively a hierarchy of up-to-date components.

This plain javascript version has been optimized for jit compilers.

In the future I might make a version that generates dom nodes instead, then executes steps in response to events, and then make it not regenerate dom nodes every step.
